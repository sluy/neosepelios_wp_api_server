Introduction

Example App

WhatsAPI comes with an example application to help you get started. The application is very simple and doesn't have all the features WhatsAPI can offer. We do not recommend to use the example app in a production environment. You can use it as a base to create your own project or just to understand how to use WhatsAPI.

Watch Demonstration Vídeo

The Example App is shipped in a .zip inside the file you received and is built with TALL Stack.
